FAQ

Est-ce que c'est vraiment gratuit ?

Oui, KickNoSub permet de regarder les VODs sub-only sans abonnement.

Comment ça marche ?

Récupérez simplement le lien de la VOD sur Kick, collez-le ici et regardez avec le chat synchronisé !

Le chat fonctionne-t-il sur les rediffusions ?

Oui, le chat est disponible si la VOD a été enregistrée avec son chat sur Kick. Notez que certaines très anciennes rediffusions n'ont pas toujours d'historique de chat conservé.

Puis-je télécharger une VOD ?

KickNoSub est un lecteur en streaming. Pour télécharger les vidéos, vous aurez besoin d'une extension tierce compatible avec le format HLS (les liens .m3u8 que nous utilisons).

L'application fonctionne-t-elle sur mobile ?

Absolument ! L'interface est 100% adaptée aux smartphones et tablettes. Vous pouvez même l'installer comme une vraie application (PWA) via le menu "Ajouter à l'écran d'accueil" de votre navigateur.

Comment marche la sauvegarde de ma progression ?

Votre progression sur chaque VOD est enregistrée automatiquement dans votre navigateur localement. Si vous fermez la page ou changez de vidéo, la lecture reprendra là où vous vous étiez arrêté.

Télécharger la VOD

Les rediffusions Kick sont diffusées en direct via un flux M3U8. Pour télécharger le fichier vidéo complet (MP4), choisissez l'une des méthodes ci-dessous :

Option 1 : Lien M3U8 Direct (Recommandé)

À utiliser avec un logiciel comme VLC (Média > Convertir/Enregistrer) ou JDownloader sur PC.

Sur Mobile : Installez l'application gratuite "1DM" (Android) ou "VLC" et collez-y ce lien.

Option 2 : Ligne de commande FFmpeg

Pour les utilisateurs avancés. Exécutez cette commande dans votre terminal PC.
